# SQL E-commerce Analysis

This project analyzes sales data from a simulated e-commerce database.

## 📁 Database Structure

The database includes:

- `customers`: customer information
- `products`: product catalog
- `orders`: transactions data

## 📊 Business Questions Answered

### 1. Total Revenue
Calculated total revenue using product price and quantity.

### 2. Revenue by Customer
Identified how much each customer spent.

### 3. Best-Selling Products
Ranked products by total units sold.

### 4. Revenue by Country
Analyzed which countries generate more revenue.

### 5. Customer Order Frequency
Measured how many orders each customer made.

## Key Insights

- Juan is the highest spending customer
- Mouse is the most sold product
- Argentina generates the highest revenue

## Skills Demonstrated

- SQL JOINs
- Aggregations (SUM, COUNT)
- GROUP BY
- Business-oriented analysis
